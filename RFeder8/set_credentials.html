set_credentials {tools}R Documentation

Set user credentials on the local credential store Using this function will allow the user to enter credentials for the selected environment. In the first popup the userid should be entered , in the second popup the password (as used to enter the respective central Feder8 application)

Description

Set user credentials on the local credential store Using this function will allow the user to enter credentials for the selected environment. In the first popup the userid should be entered , in the second popup the password (as used to enter the respective central Feder8 application)

Usage

set_credentials(env)

Arguments

env

A string in uppercase for the environment i.e. "HONEUR", "HONEUR_UAT", "HONEUR_DEV", "PHEDERATION", "PHEDERATION_UAT", "PHEDERATION_DEV".

Examples

set_credentials("HONEUR")
