list_study_documents {tools}R Documentation

List all study documents for a given study

Description

List all study documents for a given study

Usage

list_study_documents(env, study_id, auth_source)

Arguments

env

A string in uppercase for the environment i.e. "HONEUR", "HONEUR_UAT", "HONEUR_DEV", "PHEDERATION", "PHEDERATION_UAT", "PHEDERATION_DEV".

study_id

integer. Study_id as can be found in the study catalogue

auth_source

A string indicating the source of authentication i.e. "webapi" or "local".

Value

The list of documents for a given notebook

Examples

list_notebook_results("HONEUR",3,notebook_uuid = "2f5c3633-d3ec-4583-bdc2-35d4409c39b8", latest_version_only = TRUE, auth_source="local")
