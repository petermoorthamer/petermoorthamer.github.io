Saves study document for a given study

Description

Saves study document for a given study

Usage

save_study_document(env, study_id, file_name, auth_source)

Arguments

env

A string in uppercase for the environment i.e. "HONEUR", "HONEUR_UAT", "HONEUR_DEV", "PHEDERATION", "PHEDERATION_UAT", "PHEDERATION_DEV".

file_name

A string indicating the file_name to be used for saving the result

auth_source

A string indicating the source of authentication i.e. "webapi" or "local".

study_id.

integer for the study_id

Examples

save_study_document("HONEUR",3,"results.csv","local")
