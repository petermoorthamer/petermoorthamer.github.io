save_data_files {tools}R Documentation

Saves a file to the storage service

Description

Saves a file to the storage service

Usage

save_data_files(env, file_name, auth_source)

Arguments

env

A string in uppercase for the environment i.e. "HONEUR", "HONEUR_UAT", "HONEUR_DEV", "PHEDERATION", "PHEDERATION_UAT", "PHEDERATION_DEV".

file_name

A string indicating the path for the file to be saved

auth_source

A string indicating the source of authentication i.e. "webapi" or "local".

Examples

save_data_files("HONEUR","//Users/paul/results.csv","local")
