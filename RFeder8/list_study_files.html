list_study_files {tools}R Documentation

List all study related files for a given study

Description

List all study related files for a given study

Usage

list_study_files(env, study_id, auth_source)

Arguments

env

A string in uppercase for the environment i.e. "HONEUR", "HONEUR_UAT", "HONEUR_DEV", "PHEDERATION", "PHEDERATION_UAT", "PHEDERATION_DEV".

study_id

Integer representing the id for that study (can be retrieved from the study catalogue)

auth_source

A string indicating the source of authentication i.e. "webapi" or "local".

Value

The list of all study files with the uuid, timestamp and filename

Examples

list_study_files("HONEUR",3, "local")
