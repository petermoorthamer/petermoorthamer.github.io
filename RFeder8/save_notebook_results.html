save_notebook_results {tools}R Documentation

Saves results for a given notebook

Description

Saves results for a given notebook

Usage

save_notebook_results(env, notebook_uuid, file_name, auth_source)

Arguments

env

A string in uppercase for the environment i.e. "HONEUR", "HONEUR_UAT", "HONEUR_DEV", "PHEDERATION", "PHEDERATION_UAT", "PHEDERATION_DEV".

file_name

A string indicating the path to the file containing the result

auth_source

A string indicating the source of authentication i.e. "webapi" or "local".

notebook_uuid.

String with the uuid for the respective notebook that generated the results

Examples

save_notebook_results("HONEUR","2f5c3633-d3ec-4583-bdc2-35d4409c39b8","results.csv","local")
