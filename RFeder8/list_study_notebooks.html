list_study_notebooks {tools}R Documentation

List all notebooks for a given study

Description

List all notebooks for a given study

Usage

list_study_notebooks(env, study_id, external_notebook_uuid = NULL, auth_source)

Arguments

env

A string in uppercase for the environment i.e. "HONEUR", "HONEUR_UAT", "HONEUR_DEV", "PHEDERATION", "PHEDERATION_UAT", "PHEDERATION_DEV".

study_id

integer. Study_id as can be found in the study catalogue

external_notebook_uuid

String. uuid of the external notebook

auth_source

A string indicating the source of authentication i.e. "webapi" or "local".

Value

The list of notebooks for a given study including uuid, originalFilename,lastModified, key, category, uploadedByUser, uploadedByOrganizationIds, uploadDateTime

Examples

list_study_notebooks("HONEUR",3,auth_source = "local")

list_study_notebooks("HONEUR",3,auth_source = "local")[[7]]

list_study_notebooks("HONEUR",3,auth_source = "local")[[7]]$key
