Delete a file

Description

Delete a file

Usage

delete_file(env, uuid, auth_source)

Arguments

env

A string in uppercase for the environment i.e. "HONEUR", "HONEUR_UAT", "HONEUR_DEV", "PHEDERATION", "PHEDERATION_UAT", "PHEDERATION_DEV".
uuid

UUID for the file - can be found through the study_catalogue
auth_source

A string indicating the source of authentication i.e. "webapi" or "local".

Examples

delete_file("HONEUR", "2f5c3633-d3ec-4583-bdc2-35d4409c39b8", "local")
