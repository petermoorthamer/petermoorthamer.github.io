list_notebook_results {tools}R Documentation

List all results for a notebook.

Description

List all results for a notebook.

Usage

list_notebook_results(
  env,
  study_id,
  external_notebook_id = NULL,
  notebook_uuid = NULL,
  latest_version_only = FALSE,
  reverse_order = FALSE,
  auth_source
)

Arguments

env

A string in uppercase for the environment i.e. "HONEUR", "HONEUR_UAT", "HONEUR_DEV", "PHEDERATION", "PHEDERATION_UAT", "PHEDERATION_DEV".

notebook_uuid

String with the uuid of an internal notebook as found on the study cataloguye

latest_version_only

TRUE/FALSE to respectively only retrieve the latest version of every single result per source for a given notebook or retrieve all results

reverse_order

TRUE/FALSE. If set to TRUE latest files will be on top, if set to FALSE oldest files will appear on top

auth_source

A string indicating the source of authentication i.e. "webapi" or "local".

external_notebook_uuid

String with the uuid of an external notebook

Value

The list of the result files for a given notebook

Examples

list_notebook_results("HONEUR",3,notebook_uuid = "2f5c3633-d3ec-4583-bdc2-35d4409c39b8", latest_version_only = TRUE, auth_source="local")
