Get token to access files or notebooks in the storage service from the local credentialstore Credentials for the resepective credential store should be provided through set_credentials(env)

Description

Get token to access files or notebooks in the storage service from the local credentialstore Credentials for the resepective credential store should be provided through set_credentials(env)

Usage

get_token_context_from_vault(env)

Arguments

env

A string in uppercase for the environment i.e. "HONEUR", "HONEUR_UAT", "HONEUR_DEV", "PHEDERATION", "PHEDERATION_UAT", "PHEDERATION_DEV".

Examples

get_token_context_from_webapi("HONEUR")
