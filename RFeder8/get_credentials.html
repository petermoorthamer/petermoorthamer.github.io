Retrieve user credentials from the local credential store

Description

Retrieve user credentials from the local credential store

Usage

get_credentials(env)

Arguments

env

A string in uppercase for the environment i.e. "HONEUR", "HONEUR_UAT", "HONEUR_DEV", "PHEDERATION", "PHEDERATION_UAT", "PHEDERATION_DEV".

Examples

get_credentials("HONEUR")
